# Password Generator
A clean and user-friendly GUI password generator built with Python and CustomTkinter.
Generate secure, random passwords with letters, numbers, and symbols.

## Features
- Choose password length
- Select password type: Letters, Letters + Numbers, Letters + Numbers + Symbols
- Generate and copy passwords to clipboard
- Password displayed clearly in the GUI

## Requirements
- Python 3.x
- [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)

## Installation
Follow these steps in your terminal:

1. Clone the repository  
git clone https://github.com/h3r0h1/Password-Generator.git

2. Navigate to the project folder  
cd Password-Generator

3. Install requirements  
pip install customtkinter

4. Run the app  
python password_generator.py

## Usage
1. Set the desired password length in the entry field.
2. Choose the password type from the dropdown menu.
3. Click **Generate Password** to create a password.
4. Click **Copy to Clipboard** to copy the password.
